# 🎮 Jokenpo - Pedra, Papel e Tesoura

Jogo interativo de Pedra, Papel e Tesoura implementado em Python com lógica condicional e loops.

---

## 📋 Sobre o Projeto

Este é um projeto prático desenvolvido durante o **Curso em Vídeo - Python 3 (Guanabara)** para praticar:
- Estruturas condicionais (`if/elif/else`)
- Loops (`while` com `break`)
- Entrada/saída de dados (`input/print`)
- Tratamento de exceções (`try/except`)
- Listas e indexação

---

## 🚀 Como Rodar

### Pré-requisitos
- Python 3.6+ instalado

### Executar o jogo

```bash
python jokenpo.py
```

Ou:

```bash
python3 jokenpo.py
```

---

## 🎯 Conceitos Praticados

✅ **Estruturas Condicionais** - Decisões com `if/elif/else`  
✅ **Loops** - Repetição com `while True` e `break`  
✅ **Tratamento de Erros** - `try/except` para validação  
✅ **Listas** - Armazenar opções e acessar por índice  
✅ **Funções Built-in** - `input()`, `print()`, `random.choice()`  

---

## 💡 O que Aprender Aqui

Este projeto demonstra boas práticas para iniciantes em Python:
- Como estruturar um programa com entrada/processamento/saída
- Validação de entrada do usuário
- Uso de bibliotecas padrão (como `random`)
- Implementação de lógica de jogo simples
